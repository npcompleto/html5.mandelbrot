Html5.mandelbrot

Mandelbrot Set designed inside an interactive html5 canvas tag

Download this project as a .zip file Download this project as a tar.gz file
Click to center.
Press '+' key to increase the number of iterations (means more precision).
Press '-' key to decrease the number of iterations (means less precision).
Press 's' key to generate a png image from canvas.
